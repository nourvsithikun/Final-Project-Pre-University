# Fonts

Included files:

- `game_font.ttf`: DejaVu Sans for readable gameplay words, instructions, and values.
- `display_font.ttf`: DejaVu Serif Bold for Angkor-style headings.
- `DEJAVU_LICENSE.txt`: required DejaVu/Bitstream Vera license notice.

`khmer_font.ttf` remains optional. The regular game font is used when it is absent.
